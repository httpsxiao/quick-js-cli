## quick-js-cli

用于快速开发 jssdk 和 jsdemo 使用的脚手架

### Install

`npm install -g quick-js-cli`

### Usage

`quick init [name] [option]`

### Option

* -d --default, 使用默认(babel + eslint)

* -d --force, 如文件存在强制删除

* -r --registry, 设置 npm 源

### Help

```bash
quick -h
# or
quick init -h
```

### Support

* babel

* eslint

* webpack
